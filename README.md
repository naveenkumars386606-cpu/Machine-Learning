# Netflix Recommendation Engine

## Overview
A machine learning recommendation system that suggests Netflix movies and TV shows based on content similarity.

## Features
- Data preprocessing
- Feature engineering
- TF-IDF Vectorization
- Cosine Similarity
- Top-N Recommendations

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows Dataset

## How It Works
1. Load Netflix dataset
2. Clean and preprocess data
3. Convert text into vectors using TF-IDF
4. Calculate cosine similarity
5. Recommend similar movies/shows
